## 🚀 Comment Utiliser

### Lancer une session avec un agent spécifique:
```bash
# Avec Aiman (développement)
openclaw session --agent aiman

# Avec Junior (gestion projet)
openclaw session --agent eqdom-consultant

# Avec Henry (par défaut, généraliste)
openclaw session --agent main
```

### Spawn une tâche isolée:
```bash
# Tâche dev spécifique
openclaw spawn --agent aiman "Implémenter le 3-option display pour Eqdom"

# Tâche PM
openclaw spawn --agent eqdom-consultant "Créer le planning DSSi submission"
```

### Depuis une session active, switcher:
```
@app aiman "Code-moi cette feature"
@app junior "Organise mes priorités"
```
